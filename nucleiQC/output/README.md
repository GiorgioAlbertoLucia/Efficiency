# How to read the names of the files

* LHC*: name of the input dataset
* rafaelCuts: cuts on the tracks from Rafael Manhart's QC: https://its.cern.ch/jira/browse/O2-5373?page=com.docminer.jira.issue-links:com.kintosoft.jira.links.tab-panel
* yAB: rapidity cut in -A.B < y < A.B
* yToggleOn: dataset produced with the toggleOn configuration for the rapidity cut (meaning that the cut is applied to both numerator and denominator)
